## Spacing Scale

Standard 8-point scale used throughout the project:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding |
| sm | 8px | Compact element spacing, inline gaps |
| md | 16px | Default element and card padding |
| lg | 24px | Section padding, card gaps |
| xl | 32px | Layout column gaps |
| 2xl | 48px | Major section breaks (space-y-12 pattern) |
| 3xl | 64px | Page-level top spacing |

Exceptions:
- Delta indicator chips: 4px vertical, 8px horizontal padding
- Pillar comparison row: 12px vertical gap between rows (3-unit gap between 8-point anchors is acceptable for dense comparison tables)
- Touch targets for reassessment type selector buttons: minimum 44px height

Source: project convention confirmed in `EngagementDashboard.tsx` (`space-y-12`), `TrendCard.tsx` (`gap-1`, `mt-4`, `py-10`)

## New UI Surface Map

### 1. Pillar Delta Comparison Panel

Location: Client detail view (advisor), client dashboard (client)
Layout: Responsive grid — 2 columns on lg, 1 column on sm
Per-pillar card structure:
```
[Pillar name kicker]
[Previous score] → [Current score]  [+22 chip / No change]
Why it improved:
  • Governance Charter completed
  • Decision authority documented
```

- Delta chip: rounded-full, 4px vertical + 8px horizontal padding, 12px semibold
  - Positive: `bg-emerald-50 text-emerald-700 dark:bg-emerald-950 dark:text-emerald-300`
  - Negative: `bg-destructive/10 text-destructive`
  - No change: `bg-muted text-muted-foreground`
- Attribution list: 12px, muted-foreground, `•` bullet, max 3 items visible, "Show more" link if >3
- Card with `border-border/80` and `bg-card/92`
- Zero-state card text: "No change — No new planning activity" centered in card body

### 2. Reassessment Type Selector (Dialog)

Triggered by: "Start Reassessment" button on client detail / dashboard
Three option cards in a vertical list within Dialog:
- Full Assessment — description: "Annual review or major life event"
- Domain / Pillar Reassessment — description: "Retake a single pillar"
  - Pillar selector appears inline when chosen (Select component)
- Targeted Follow-up — description: "Re-ask questions linked to completed recommendations"
  - Count badge: "3 eligible questions" shown inline

Selected option: left border accent (2px solid `--accent`) + `bg-accent/5`
CTA at bottom: "Begin Reassessment" (primary button)
Disclaimer below CTA: 12px muted-foreground "Previous assessment is preserved — this starts a new version."

### 3. Review Cadence Panel

Location: Advisor client detail view, advisor cadence settings
Shows: Current cadence, next due date, overdue indicator
Cadence states:
- On track: `--chart-2` green badge "On track"
- Due soon (≤14 days): `--accent` amber badge "Due in N days"
- Overdue: `--destructive` red badge "Overdue — N days past due"
- System recommended: accent badge "AKILI Recommends" with action button "View Recommendation"

Override control: "Override cadence" link → opens popover with Select (Annual / Semi-annual / Quarterly) + Save button

### 4. Intelligence Timeline (evolved from Phase 23 activity feed)

Location: Advisor client detail view, client dashboard activity section
Tab filter: All | Assessments | Score Changes | Cadence | Recommendation Impact
Each event row:
- Icon (16px, lucide) + event summary text (14px) + relative timestamp (12px muted)
- Assessment events: `FileText` icon
- Score change events: `TrendingUp` / `TrendingDown` icons
- Cadence events: `Calendar` icon
- Recommendation impact events: `CheckCircle2` icon

No change from Phase 23 data model — new event types appear automatically in the feed with new action strings.

## Interaction States

| State | Treatment |
|-------|-----------|
| Loading (delta cards) | Skeleton cards: same grid layout, `animate-pulse` via shadcn Skeleton |
| Loading (timeline) | 5 skeleton rows at 40px height |
| Error (score fetch) | Inline error within card: "Could not load scores. Retry." with `Button variant="ghost"` |
| Empty (no previous assessment) | Full-width dashed border card: "Complete your first reassessment to see improvement trends." + "Start Reassessment" CTA |
| Empty (timeline, filtered tab) | Centered muted text: "No [tab name] events yet." |
| Disabled (targeted follow-up, no eligible questions) | Option card shown but grayed (`opacity-50`, `cursor-not-allowed`), tooltip: "No completed recommendations with linked questions yet" |
