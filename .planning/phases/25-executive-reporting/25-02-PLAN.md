  <action>
Create GET route handler following the exact pattern from src/app/api/reports/by-id/[reportId]/pdf/route.tsx.

Function signature: export async function GET(_request: NextRequest, { params }: { params: Promise<{ reportId: string }> })

Auth flow (three-bucket, identical to analog):
1. const session = await auth(); if (!session?.user?.id) return 401
2. const { reportId } = await params
3. Fetch ExecutiveReport by id, include: { client: { select: { id: true } }, advisorProfile: { select: { id: true, userId: true } } }
4. If not found, return 404
5. Determine auth bucket:
   - isOwner = report.client.id === session.user.id
   - isAdmin = isPlatformAdminRole(session.user.role)
   - isAssignedAdvisor: if ADVISOR role, check advisorProfile.userId === session.user.id
   - If none: return 403
6. Client access restriction: if isOwner and report.status === "DRAFT", return 403 (clients cannot see drafts per D-21)

Variant determination:
- URL query param: ?variant=advisor (default: "client")
- If variant === "advisor" and isOwner (client user), return 403 (clients cannot access Advisor Brief per D-17)

Rendering:
- PUBLISHED or SUPERSEDED: use frozen executiveSnapshotData and brandingSnapshot from the DB row. Parse executiveSnapshotData as ExecutiveReportSnapshot. Overlay advisorNotes, meetingAgenda, discussionPrompts from DB row onto snapshot (they were frozen at publish time).
- DRAFT: call buildExecutiveReportSnapshot(report.clientId, report.advisorProfileId) for live data. Call buildBrandingSnapshot using report's latest assessment. Overlay advisorNotes/meetingAgenda/discussionPrompts from the DRAFT row.
- Call renderExecutiveReportPdf({ snapshot, branding, variant, draft: report.status === "DRAFT" })

Response headers (same as analog):
- Content-Type: application/pdf
- Content-Disposition: attachment; filename="${firmSlug}-executive-report-v${report.version}.pdf" (or "-advisor-brief-v${version}.pdf" for advisor variant)
- Cache-Control: no-cache, no-store, must-revalidate
- Pragma: no-cache, Expires: 0

Error handling: catch all, console.error, return 500 with generic message.
